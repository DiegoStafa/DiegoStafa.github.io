Limitatezza di una successione

ipotesi: * lim an -> L ∀ n ≥ n0

tesi: * an è una successione limitata

dimostrazione 1. si scrive la definizione di limite che tende a L 2. ∀ n ≥ n0 la successione è limitata per definizione del limite 3. ∀ n < n0 la successione è limitata perchè ha un numero finito di elementi 4. an è limitata perchè è l'unione di due insiemi limitati

Teorema dell'unicità del limite

ipotesi: * lim an -> L ∀ n ≥ n0

tesi: * L è unico

dimostrazione per assurdo 1. si scrive la definizione di limite per L1 2. si scrive la definizione di limite per L2 3. si impone un n0 a partire dal quale 1) e 2) sono verificate 4. si usa la disuguaglianza triangolare 5. si arriva a 0 ≤ |L1 - L2| ≤ epsilon 6. si arriva ad L1 - L2 = 0 7. assurdo dimostrato


Teorema della permanenza del segno

ipotesi: * lim an -> L ∀ n ≥ n0

tesi: 1. se L > 0 allora an ≥ 0 ∀ n ≥ n0 2. se L < 0 allora an ≤ 0 ∀ n ≥ n0 3. in generale se L diverso da 0 allora |an| > |L/2|0 ∀ n ≥ n0

dimostrazione 1) 1. si scrive la definizione di lim an -> L, cioè: |an - L| ≤ epsilon 2. si sceglie epsilon uguale a L/2 e si ha si ha |an| ≤ L/2 3. si scrive il valore assoluto come intervallo 4. si porta L fuori dal valore assoluto 5. si ha in particolare che L/2 ≤ an

dimostrazione 2) uguale alla 1) solo che si scegli epsilon uguale a -L/2

dimostrazione 3) è solo l'unione dei casi precedenti


Teorema dei due carabinieri

ipotesi: * an ≤ cn ≤ bn * lim an = lim an = L

tesi: * lim an = L

dimostrazione 1. si scrive la definizione di lim an = L 2. si scrive la definizione di lim bn = L 3. si impone un n0 a partire dal quale 1) e 2) sono verificate 4. si ha che L - epsilon ≤ an ≤ L + epsilon 5. si ha anche che L - epsilon ≤ bn ≤ L + epsilon 5. dall'ipotesi L - epsilon ≤ an ≤ cn ≤ bn ≤ L + epsilon 6. quindi in particolare L - epsilon ≤ cn ≤ L + epsilon
