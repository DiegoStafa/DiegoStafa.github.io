successione convergente a 0

una successione si dice convergente a 0 se: * lim an -> 0

cioè:

  • ∀ epsilon > 0 ∃ n0 : ∀ n ≥ n0 si ha che |an| ≤ epsilon

si legge: per ogni epsilon maggiore di 0, esiste un n0 tale per cui: per ogni n maggiore o uguale a n0 si ha che il valore assoluto della successione è minore di epsilon


successione divergente a + infinito

una successione si dice divergente a + infinito se:

  • lim an -> +inf

cioè:

  • ∀ M > 0 ∃ n0 : ∀ n ≥ n0 si ha che an ≥ M

si legge: per ogni ’M' maggiore di 0, esiste un n0 tale per cui: per ogni n maggiore o uguale a n0 si ha che la successione è maggiore di ’M'


successione convergente a L

una successione si dice convergente a L se:

  • lim an - L -> 0

cioè:

  • ∀ epsilon > 0 ∃ n0 : ∀ n ≥ n0 si ha che |an - L| ≤ epsilon

e si indica con lim an -> L

si legge: per ogni epsilon maggiore di 0, esiste un n0 tale per cui: per ogni n maggiore o uguale a n0 si ha che il valore assoluto della successione meno ‘L’ è minore di epsilon

proprietà successioni convergenti/divergenti (1)

ipotesi: * lim an -> +inf

tesi: * lim 1/(an) -> 0


proprietà successioni convergenti/divergenti (2)

ipotesi: * lim an -> 0

tesi: * lim |(an| -> 0


proprietà successioni convergenti/divergenti (3)

ipotesi: * lim -an -> +inf

tesi: * lim an -> -inf


proprietà successioni convergenti/divergenti (4)

ipotesi: * |an| ≤ M * lim bn -> 0

tesi: * lim an * bn -> 0


Aritmetica dei limiti

ipotesi: * lim an -> A * lim bn -> B

tesi: * lim an + bn -> A + B * lim an - bn -> A - B * lim an * bn -> A * B * lim an / bn -> A / B


Criterio del rapporto

ipotesi: * an ≥ 0 * lim an+1 / an -> L

tesi: * se L > 1 allora lim an -> +inf * se L < 1 allora lim an -> 0 * L = 1 non da informazioni
